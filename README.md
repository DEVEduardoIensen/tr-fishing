# 🎣 TR Fishing — A Marca do Pescador

> Landing Page e Catálogo Interativo Premium para a marca **TR Fishing**, desenvolvedora de equipamentos de alta performance para pesca esportiva.

Este repositório contém o código-fonte do portal oficial da **TR Fishing**, projetado para oferecer uma experiência digital imersiva, moderna e de alta fidelidade visual. O site apresenta a marca, seus valores, e disponibiliza um catálogo completo e filtrável de linhas multifilamento, varas de carbono IM7 e iscas soft premium enriquecidas com sal e essência de alho.

---

## ✨ Funcionalidades Principais

* **Design Glassmórfico & Premium**: Interface escura e moderna com paletas de cores tailoreadas (laranja vibrante e azul profundo), gradientes fluidos e contrastes de alto padrão.
* **Catálogo Interativo Dinâmico**: Visualização de produtos em destaque na página principal, com filtros suaves em tempo real e abertura de especificações técnicas em um modal flutuante.
* **Efeitos Visuais Premium**: Efeito 3D de inclinação (tilt) e reflexo (glare) interativo ao passar o mouse sobre os cards de produtos.
* **Ajustes de Responsividade Finos**: Adaptado de forma inteligente para Mobile e Tablet, ocultando elementos desnecessários (como formulários extensos e botão de retorno ao topo) e reorganizando o menu de hambúrguer e rodapé para a melhor legibilidade e usabilidade.
* **Integração Comercial Inteligente**: Formulário e botões de catálogo parametrizados para redirecionar o usuário diretamente para o WhatsApp comercial com mensagens estruturadas prontas.
* **SEO e Acessibilidade**: Tags de metadados completas para indexação em motores de busca (Google) e compartilhamento em mídias sociais (Open Graph), além de semântica HTML5.

---

## 🛠️ Tecnologias Utilizadas

O portal foi construído utilizando tecnologias puras da Web para máxima performance de carregamento, sem a necessidade de frameworks pesados:

* **HTML5**: Estrutura semântica e SEO-friendly.
* **CSS3 (Vanilla)**: Design System completo baseado em variáveis (custom properties), flexbox, CSS Grid e animações fluidas.
* **JavaScript (Vanilla)**: Manipulação do DOM para abertura de modais, filtragem do catálogo, manipulação do carrossel e efeitos 3D responsivos via Intersection Observer.

## 🚀 Como Executar Localmente

Como o projeto é construído em código estático puro (HTML/CSS/JS), você não precisa instalar dependências complexas.

1. **Clonar o Repositório**:
   ```bash
   git clone https://github.com/DEVEduardoIensen/tr-fishing.git
   ```
2. **Abrir o Projeto**:
   Navegue até a pasta do projeto e abra o arquivo `index.html` diretamente no seu navegador, ou utilize extensões como o **Live Server** (do VS Code) para rodar um servidor local em `http://localhost:5500`.
